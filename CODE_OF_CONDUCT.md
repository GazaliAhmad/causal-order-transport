# Code of Conduct

## Our Commitment

This project is committed to providing a respectful, professional, and harassment-free environment for everyone participating in the repository and related community spaces.

## Expected Behavior

Participants are expected to:

- communicate respectfully and constructively
- assume good intent while giving clear technical feedback
- focus discussion on the work, not on personal attacks
- welcome differing viewpoints and experience levels
- help keep the project useful and safe for others

## Unacceptable Behavior

The following behavior is not acceptable:

- harassment, intimidation, or discrimination
- abusive, insulting, or demeaning language
- personal attacks or bad-faith provocation
- publishing private information without permission
- disruptive conduct that repeatedly derails technical discussion

## Scope

This Code of Conduct applies to:

- repository issues
- pull requests
- discussions
- review comments
- other project-managed communication spaces

## Enforcement

Project maintainers may remove, edit, or reject comments, contributions, or participation that violate this Code of Conduct.

Repeated or serious violations may result in temporary or permanent restrictions from participation in the project.

## Reporting

If you experience or witness behavior that violates this Code of Conduct, contact the project maintainer through the repository contact channels listed in the package metadata or repository profile.

## Maintainer Standard

Maintainers are expected to enforce this Code of Conduct consistently, fairly, and with attention to context and participant safety.
